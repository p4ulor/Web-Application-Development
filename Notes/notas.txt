O Spring (framework) desenvolveu 3 principais subprojetos

- Spring frame work também é de chamado de spring context. É a componente base de muitos outros projetos. Implementa um contentor de injeçao de dependencias
- spring web = mvc -> framework para criaçao de applicaçoes WEB, que usa o spring context
- spring reactive web = web flux -> alternativa ao spring MVC

@Bean -> objeto gerido pelo contexto
